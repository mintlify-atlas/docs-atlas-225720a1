## List Vehicles

`GET /api/vehicles`

Retrieve all vehicles, optionally filtered by collection center. Results are sorted with default vehicles first, then by plate.

### Query Parameters

<ParamField query="collectionCenterId" type="string">
  Filter vehicles by collection center ID. If not provided, uses the active center from configuration.
</ParamField>

### Response

Returns an array of vehicle objects sorted by `is_default DESC, plate ASC`.

<Expandable title="Vehicle Object">
  <ResponseField name="id" type="string">
    Unique vehicle identifier
  </ResponseField>
  <ResponseField name="plate" type="string">
    License plate (unique)
  </ResponseField>
  <ResponseField name="brand" type="string">
    Vehicle brand/manufacturer
  </ResponseField>
  <ResponseField name="model" type="string">
    Vehicle model
  </ResponseField>
  <ResponseField name="owner" type="string">
    Vehicle owner name
  </ResponseField>
  <ResponseField name="is_default" type="boolean">
    Whether this is the default vehicle for the collection center
  </ResponseField>
  <ResponseField name="collection_center_id" type="string | null">
    Associated collection center ID
  </ResponseField>
  <ResponseField name="created_at" type="string">
    Timestamp when vehicle was created
  </ResponseField>
</Expandable>

### Example Request

```bash
curl http://localhost:4000/api/vehicles?collectionCenterId=center-1
```

### Example Response


## Create Vehicle

`POST /api/vehicles`

Create a new vehicle. If `isDefault` is true, all other vehicles for the same collection center will be set to non-default.

### Request Body

<ParamField body="id" type="string" required>
  Unique vehicle identifier (UUID)
</ParamField>

<ParamField body="plate" type="string" required>
  License plate (must be unique across all vehicles)
</ParamField>

<ParamField body="brand" type="string" required>
  Vehicle brand/manufacturer
</ParamField>

<ParamField body="model" type="string" required>
  Vehicle model
</ParamField>

<ParamField body="owner" type="string" required>
  Vehicle owner name
</ParamField>

<ParamField body="isDefault" type="boolean" default="false">
  Set as default vehicle. If true, unsets all other vehicles as default for the same center.
</ParamField>

<ParamField body="collectionCenterId" type="string">
  Associated collection center ID (optional)
</ParamField>

### Response

Returns the created vehicle object with `created_at` timestamp.

<Note>
  The operation runs in a transaction. If `isDefault` is true, it will:
  1. BEGIN transaction
  2. Set all other vehicles to `is_default = false` for the same center
  3. Insert the new vehicle with `is_default = true`
  4. COMMIT transaction
</Note>

### Example Request

```bash
curl -X POST http://localhost:4000/api/vehicles \
  -H "Content-Type: application/json" \
  -d '{
    "id": "vehicle-new-1",
    "plate": "DEF-456",
    "brand": "Chevrolet",
    "model": "LUV D-MAX 2021",
    "owner": "María García",
    "isDefault": false,
    "collectionCenterId": "center-1"
  }'
```

### Example Response

```json
{
  "id": "vehicle-new-1",
  "plate": "DEF-456",
  "brand": "Chevrolet",
  "model": "LUV D-MAX 2021",
  "owner": "María García",
  "is_default": false,
  "collection_center_id": "center-1",
  "created_at": "2026-03-09T16:00:00Z"
}
```

## Update Vehicle

`PUT /api/vehicles/:id`

Update an existing vehicle. If `isDefault` is set to true, all other vehicles for the same collection center will be set to non-default.

### Path Parameters

<ParamField path="id" type="string" required>
  Vehicle ID
</ParamField>

### Request Body

<ParamField body="plate" type="string" required>
  License plate
</ParamField>

<ParamField body="brand" type="string" required>
  Vehicle brand
</ParamField>

<ParamField body="model" type="string" required>
  Vehicle model
</ParamField>

<ParamField body="owner" type="string" required>
  Vehicle owner name
</ParamField>

<ParamField body="isDefault" type="boolean" required>
  Set as default vehicle
</ParamField>

<ParamField body="collectionCenterId" type="string">
  Associated collection center ID
</ParamField>

### Response

Returns the updated vehicle object.

<Note>
  The update runs in a transaction. If `isDefault` is true, it will:
  1. BEGIN transaction
  2. Set all other vehicles (excluding this one) to `is_default = false`
  3. Update this vehicle with the new values
  4. COMMIT transaction
</Note>

### Example Request

```bash
curl -X PUT http://localhost:4000/api/vehicles/vehicle-1 \
  -H "Content-Type: application/json" \
  -d '{
    "plate": "ABC-123",
    "brand": "Toyota",
    "model": "Hilux 2022",
    "owner": "Alternativa Verde C.A.",
    "isDefault": true,
    "collectionCenterId": "center-1"
  }'
```

### Example Response

```json
{
  "id": "vehicle-1",
  "plate": "ABC-123",
  "brand": "Toyota",
  "model": "Hilux 2022",
  "owner": "Alternativa Verde C.A.",
  "is_default": true,
  "collection_center_id": "center-1",
  "created_at": "2026-01-10T09:00:00Z"
}
```

## Delete Vehicle

`DELETE /api/vehicles/:id`

Delete a vehicle by its ID.

### Path Parameters

<ParamField path="id" type="string" required>
  Vehicle ID
</ParamField>

### Response

<ResponseField name="ok" type="boolean">
  Returns `true` if deletion was successful
</ResponseField>

### Example Request

```bash
curl -X DELETE http://localhost:4000/api/vehicles/vehicle-2
```

### Example Response

```json
{
  "ok": true
}
```

### Error Response

```json
{
  "error": "DB delete error"
}
```
